PathFinderJS.

PathFinderJS will accept an JSON Object and a String path for the property that you wish to access.
PathFinder will return the property of type matched in the path. That could be a string a subset of the object, an array of strings or Objects. See the examples below or run in a browser where console output is supported, to see the output of the pathfinder.html page in the console.

Example: JSON Object: var response = {"something":[{"_id":"4ed8f7744ff24c5f0c000001","properties":[{"sport":[{"fb":"one"}]}],"screen_name":"daithi44",},{"_id":"4ed8f840beed7e600c000001","properties":[{"sport":[{"fb":"two"}]}],"screen_name":"daithi44"},{"_id":"4ed9020eee56c9890c000001","properties":[{"sport":[{"fb":"three"}]}],"screen_name":"daithi44"}]};
Path: 'something[1].properties[0].sport[0].fb';

Make Call to pf.pathFinder

1. pf.pathFinder(response,'something[1].properties[0].sport[0].fb');
returns '["one", "two", "three"]'

2. pf.pathFinder(response.something[2],'properties[0].sport[0].fb')
returns 'three'.

3. pf.pathFinder(response,'something');
returns an Array of 3 objects.
